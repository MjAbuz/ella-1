softuni projects

I've contributed in 11 different projects with different teams, using different technologies.

Teamwork projects are great way to learn working in team and study fast new things.

projects and contribution

Jumping Dude is C#-based Windows Forms game, with two heroes: The Dude and the enemy - Sonic. The goal of the game is to reach the Diamond on the top of the screen by avoiding getting caught by the enemy. Memory Game is a card-matching game with three themes and levels, which can be chosen by the user. We used Java and Swing. In Programmer's Dream you can run endlessly and collect 1 and 0 through the Programmer's World. bYte me is a date site for programmers. Every user has profile with likes and dislikes, and depending on the count of dislikes you can order rewards. For this project I used C#, APS.NET MVC, Entity Framework, SQL Server, HTML and CSS.

Tech Talks is JavaScript and ReactJs-based blog, when users can add posts and read other's posts. And finally, Everyday Journal is an online diary for people who are obsessed with healthy living. They can traverse through days, write what they've consumed or their tasks for the day. In the Practical Teamwork course I had 3 different small task and 1 project with different teams. We worked with Git and source control systems, we learned how to make a UI Prototyping for given web application and get to know what are these project tracking tools.
